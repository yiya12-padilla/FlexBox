EDITORES DE CODIGO

Los editores de código son las herramientas que todo desarrollador debe tener a mano. Permiten editar código fuente en diversos lenguajes de programación y ofrecen múltiples herramientas para facilitar el trabajo y aumentar la productividad. Estos editores también proveen un modo conveniente de ejecutar un compilador, un intérprete, un depurador, o cualquier otro programa que sea relevante en el proceso de desarrollo de software. Por lo que, si bien muchos editores de texto pueden ser usados para editar código fuente sin problemas, si no mejoran, automatizan y facilitan la edición del código, no ameritan ser llamados "editores de código fuente", y son únicamente editores de texto que pueden ser usados para editar código fuente. alguno de estos editores son :

SUBLIME TEXT 3

Sublime Text 3 un editor de código válido para Mac, Windows y Linux y es uno de los editores de texto más completos que existen. Es ligero, open source, con una estabilidad alta y te permite editar un archivo de forma muy rápida. Y precisamente, su facilidad de uso y lo rápido que te permite editar un archivo es lo que lo convierte en una joyita dentro de los editores de su clase. Otro de los puntos fuertes de Sublime Text es que es increíblemente extensible, ya que cuenta con una enorme cantidad de plugins disponibles para instalar. Esto multiplica las opciones, como temas con los que puedes personalizar la apariencia del editor, lint de código (un asistente para localizar muy rápido posibles errores en el código), herramientas de definición de colores… y todo tipo de funcionalidades útiles cuando trabajas con editores de código en el desarrollo o diseño de una web.

CODA

Entre los editores de código para Mac, Coda es una de las mejores opciones. Desarrollado por Panic, con este editor de código no necesitas un software FTP, así que no solo podrás trabajar con archivos locales, sino también editar de forma remota. Rápido, limpio y potente, tiene todas las funciones que esperas de una herramienta de este tipo, como función autocompletar, resaltado de sintaxis, inspector web, depurador de perfiles, plegado de código, buscar y reemplazar, cierre automático de etiquetas, guías de sangría o comentarios de los cambios de código. Al igual que Sublime Text, Coda cuenta con una amplia lista de plugins disponibles que añaden otras funcionalidades extra.

VISUAL STUDIO CODE

Gratuito, estable, open source, muy robusto y buena velocidad de trabajo son algunas de las principales características de Visual Studio Code, el editor de código desarrollado por Windows (aunque también funciona con Mac; y, además, muy bien). Especialmente recomendado para proyectos grandes y complejos. Aunque es un poco lento al arrancar, una vez que empiezas a usarlo, es rápido y te permite hacer bastantes acciones interesantes, como control Git (para que los cambios que hagas en tus archivos se resalten en varios lugares), autocompletar con IntelliSense, descripción automática en tiempo real de la API, abrir y ordenar múltiples carpetas, comparación de versión lado a lado (para tener claridad extra) y es compatible con más de 30 lenguajes. Unido a que también admite extensiones, es una buena opción para modificar código web y programar si trabajas con un equipo de 64 bits, así que si usas un sistema de 32 bits, no podrás usarlo.

ATOM

Atom podría definirse como una versión gratuita de Sublime Text con interfaz amigable. Es gratis, con buena estabilidad (aunque no mi favorita en este sentido), buena velocidad y con altas posibilidades. Se trata de un editor de texto web open source desarrollado por GitHub. Aunque en sus inicios tuvo algunos problemas de estabilidad y funcionamiento, se han ido reduciendo considerablemente a medida que el software ha madurado. Es cierto que aún es más lento que algunos de sus competidores, pero es tan útil y rápido como cualquiera del resto de opciones. Esto, unido a su carácter gratuito, lo convierten en uno de los editores de código que tener en la recámara.

BRACKETS

brackets es una interesante herramienta con la que los desarrolladores y diseñadores podemos trabajar HTML, CSS, JavaScript y otros muchos lenguajes. Es gratuito y aunque las opciones de personalización no son tan amplias como otros ya mencionados, la estabilidad, rapidez y posibilidades de trabajo son más que aceptables, gracias a que se le pueden añadir plugins que aumentan sus funcionalidades. Desarrollado por Adobe y disponible para Windows, Mac y Linux, es también un editor open source. Lleva pocos años en el mercado y, por tanto, aún le queda recorrido de maduración, pero parece que es una buena alternativa.
